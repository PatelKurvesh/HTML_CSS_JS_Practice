HTML

  • HTML stands for Hyper Text Markup Language.

  • HTML is a markup language that tells web browsers how to structure the web pages you visit.

  • HTML is a Client-side

CSS

  • CSS stands for Cascading Style Sheet.

  • CSS is a style sheet language used to describe the presentation of a document written in HTML or XML etc.

  • CSS is used to style web pages, define the layout of page elements and create visual effects such as color, typography, and animation.

JavaScript

  • JavaScript is a popular scripting language.

  • JavaScript is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions.

  • JS can update and change both HTML and CSS.
